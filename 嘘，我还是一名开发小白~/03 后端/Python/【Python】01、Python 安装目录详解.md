## Python 安装目录

- **DLLs**

  python 的动态库。

- **doc**

  python 的说明文档。

- **include**

  包含共享目录。

- **lib**

  python 的标准库。

- **libs**

  python 的内置库。

- **Scripts**

  python 的各种脚本。

- **Tcl**

  python 内置的 GUI 工具

- **Tools**

  python 提供的一些工具。

> ***💬说明：*** *以上基于 Python39 为准。*
