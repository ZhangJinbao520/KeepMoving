## 文档规范

1. 同级不换行

2. 降级不换行

3. 升级换 1 行

4. font 元素只用于定义字体颜色

5. 说明的格式：

   ```markdown
   ***💬说明**：说明。*
   ```

6. `HTML 元素` < `==` < `~~` < `*` < `**`
